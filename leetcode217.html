Leetcode 217 - Contains Duplicate


Link: https://leetcode.com/problems/contains-duplicate/

Approach(es)



Approach 1: Start from the first element, iterate the complete array to check if it is a duplicate. If you find a duplicate at any point, return true. If not, repeat this for second array element till either you find a duplicate or you have exhausted all the array elements. If no duplicate found, return false.
Time complexity:O(n^2) because you will be running two loops.
Space complexity:O(1)

Approach 2: To optimize the solution above, you will consider optimizing time complexity from quadratic to linear or logarithmic possibly. Using a hashset can help to reduce complexity to linear time O(n) but at the cost of additional space O(n).
