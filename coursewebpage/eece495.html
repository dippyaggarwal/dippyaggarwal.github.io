Course Description:

Lab to accompany ECES 493, Software Engineering. Students learn the basics of object-oriented analysis and design, apply concepts taught in Software Engineering lecture, and develop a group project.

Prerequisites:

Introduction to programming.

Corequisite:

EECE 493: Software Engineering. It is mandatory for anyone taking this lab (EECE 495) to be enrolled in the EECE 493 as well.

Textbook:

Deitel and Deitel, Java, How to Program, Ninth Edition, ISBN 978-0-13-605306-4, Prentice-Hall, 2012, online edition available through Engineering Library. (or other Java reference which you already own). NOTE: Eighth Edition is available in Engineering Library permanent reserves.

References on Web:
